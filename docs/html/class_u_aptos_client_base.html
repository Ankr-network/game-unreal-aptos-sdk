SendRequest function is used to send GET or POST request and the response will be received as a JsonObject.

The function requires parameters described below and returns nothing.
Inside the function, A Get or POST request is sent to the specified URL.

Parameters
_urlThe URL to which the request will be sent.
_verbWhether the request is a GET or POST request.
_payloadThe payload data associated with the request
_functionThe function to be executed when a response is received.
_backgroundTaskDetermines if the request to be processed as a background task.
